Here is the Summary of the Site

  1. All the content is viewable to every user , irrespective of whether he is logged in or not
  2. Any user can view details of the trip by clicking on its title
  3. Any user can view profile of the trip creator by clicking on the username below the picture
  4. Any user can view profile of the user who has put his review or comment on the article
  5. Using rome2rio API, user can search how to reach a destination using the "How to Reach" tab
  6. Any user can search how to reach a given trip's destination from the trip's details page by just clicking the button "How To Reach"
  7. The user has to login/register to enter any data into the system
  8. To register
    1. All the fields are mandatory
    2. The user must provide a valid email id(valid format)
    3. The user must choose a non existing username
    4. The passwords entered must match
    5. Upon clicking register button, user will be intimated about the errors mentioned above, if any
  9. To Login
    1. The user must enter valid credentials else the user will be prompted to enter valid credentials
  10. Any logged in user can follow any other user
  11. All follower and following users can be viewed in the profile page
  12. A logged in user can edit only his profile
  13. Only logged in user can create a new trip or delete his trip
  14. Only logged in user can view all his trips in his profile page
  15. Only the owner of the trip can edit it or delete it
  16. Any logged in user can comment on any of the trip
